## 💜 О проекте

**Yandex Music Controller** — это лёгкое и удобное десктопное приложение для управления Яндекс Музыкой прямо с рабочего стола.

Приложение позволяет управлять воспроизведением, не открывая сам плеер, с помощью современного интерфейса и горячих клавиш.

## 🎉 Релиз приложения

🚀 Приложение **Yandex Music Controller** официально создано и опубликовано!

## 📦 Установка

### 🔹 Вариант 1 (рекомендуется)

Скачать готовый `.exe` файл из раздела **Releases** и запустить.

---

### 🔹 Вариант 2 (через Python)

```bash
git clone https://github.com/Anyusek/Yandex-music-controller.git
cd Yandex-music-controller
pip install -r requirements.txt
python yandex_music_controller.py
```

---

## ⚙️ Как пользоваться

1. Запустите приложение
2. Укажите путь к Яндекс Музыке:

   * через кнопку
   * или перетащите `.exe` / ярлык в окно
3. Нажмите **"Открыть Я.Музыку"**
4. Управляйте музыкой 🎧

---

## ⚠️ Примечания

* Работает только на Windows
* Требуется установленная Яндекс Музыка
* Python 3.10+
* Для работы горячих клавиш может потребоваться запуск от имени администратора
* При отсутствии некоторых библиотек часть функций может быть недоступна

## 🔥 Глобальные горячие клавиши

| Действие        | Клавиши  |
| --------------- | -------- |
| Пауза / Воспроизведение    | Ctrl + P |
| Следующий трек  | Ctrl + → |
| Предыдущий трек | Ctrl + ← |
| Громче          | Ctrl + ↑ |
| Тише            | Ctrl + ↓ |
| Mute            | Ctrl + M |
