# Yandex-music-controller

Лёгкое десктопное приложение для управления Яндекс Музыкой, написанное на Python с использованием CustomTkinter и Win32 API.

## ✨ Возможности

* ▶️ Воспроизведение / пауза
* ⏭ Следующий / предыдущий трек
* 🔊 Управление громкостью
* 🚀 Запуск Яндекс Музыки
* 🪟 Переключение на окно плеера
* ⌨️ Глобальные горячие клавиши
* 📌 Сворачивание в системный трей

## 🛠 Используемые технологии

* Python 3
* CustomTkinter (GUI)
* Win32 API (pywin32)
* keyboard (горячие клавиши)
* pystray (системный трей)

### 📚 Используемые модули и библиотеки

Приложение использует следующие внешние библиотеки:

* `customtkinter` — для создания графического интерфейса
* `pywin32` — для работы с Win32 API (управление окном приложения)
* `keyboard` — для глобальных горячих клавиш
* `pystray` — для работы с системным треем
* `Pillow` — для создания иконки в трее

## 🚀 Установка и запуск

```bash
git clone https://github.com/Anyusek/Yandex-music-controller.git
cd Yandex-music-controller
python yandex_music_controller.py
```
## 📦 Зависимости

Перед запуском необходимо установить все используемые библиотеки:

```bash
pip install -r requirements.txt
```

### ⚠️ Примечания

* Работает только на Windows
* Если не установлены `keyboard`, `pystray` или `Pillow`, приложение всё равно запустится, но горячие клавиши будут отключены и сворачивание в трей не будет работать
* Убедитесь, что используете Python версии 3.10 или выше
* На Windows может потребоваться запуск от имени администратора для работы горячих клавиш
* Яндекс Музыка должна быть установлена на ваш ПК

## 🔥 Глобальные горячие клавиши

* **ctrl+p** - делает паузу трека
* **ctrl+right** - переключает на следующий трек
* **ctrl+left** - переключает на предыдущий трек
* **ctrl+up** - повышает громкость
* **ctrl+down** - уменьшает громкость
* **ctrl+m** - делает заглушку (т.е. отключает звук)
